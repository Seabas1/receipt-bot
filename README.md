# Receipt Bot

Telegram-бот для распознавания чеков и создания Excel-таблиц с позициями и ценами.

## Структура проекта

receipt_bot/
├── src/ 
├── models/ 
├── data/ # Данные для обучения
│ ├── synthetic/ # Синтетические данные
│ └── real/ # Реальные чеки
├── dataset/ # Сгенерированный датасет (изображения + разметка)
├── main.py 
└── requirements.txt
